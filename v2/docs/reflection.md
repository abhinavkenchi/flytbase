# V1.1 Reflection

The biggest improvement in V1.1 was converting backend logic into measurable and visible outputs.

While V1 focused on core conflict detection, V1.1 focused on usability, validation, and scalability evidence.

The benchmark module was especially valuable because it quantified system performance under increasing drone counts. This changed the project from a conceptual prototype into a tested engineering system.

Dashboard upgrades also improved communication by making conflicts, alerts, and mission states visible in real time.

A key lesson learned was that technical improvements must be observable to create impact.
